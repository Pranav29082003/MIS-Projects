# 📦 Inventory Management & MIS Dashboard (Excel)

## 📌 Project Overview

The **Inventory Management & MIS Dashboard** is an interactive Excel dashboard designed to monitor inventory performance, stock levels, sales, supplier information, and reorder requirements. It enables business users to make informed inventory decisions through dynamic visualizations and KPI tracking.

This project demonstrates the use of **Microsoft Excel** for data cleaning, analysis, reporting, and dashboard development using Pivot Tables, Pivot Charts, Slicers, Timelines, and advanced Excel functions.

## 🎯 Business Objective

The objective of this project is to help inventory managers and business stakeholders:

- Monitor inventory levels across product categories.
- Track sales performance.
- Identify products requiring replenishment.
- Analyze supplier-wise inventory value.
- Monitor inventory trends over time.
- Support data-driven inventory planning and operational decision-making.
